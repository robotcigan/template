Introduction

Welcome to the Sport Website Template and documentation.

Thanks for purchasing this template. Please read the documentation carefully before sending a support request, the chance you could find the answer here is big and would save you a lot of time! If you're having an issue with the theme and can't find the answer here, then please send me an email through my profile page and I will try to answer you as quick as possible..

This template is not only a responsive theme meaning it is optimised for multiple device sizes, it is also retina ready.

Installation

To install this template onto your web server, you need to transfer the web files up to the root directory of your web host. Either use an FTP program like Filezilla, or upload files through the cpanel of your hosting account.

This is a HTML template, not installable into CMS's like Wordpress, etc

Note:

Filezilla is a free FTP program, available here: Filezilla

Changing Logo's

Images and logo's are found in the "img" folder. Simply replace the logo files with your own, it's easiest to just overwrite the existing logo files with your own logo. Remember to use HiDPI versions.

Because this theme is Retina ready, and Retina Images are downscaled for Retina screens, you may have to update the size of the allocated area for your logo to avoid distortion or cropping. To do this open the styles.css file and look for

/* ===================================================
   Logos
================================================ */

Here you can see max width properties. Match the height and width to suite the size of your logo when halved or divided by 2.

Example:

You will see that the provided logo is actually 302px wide, however we half the size in our CSS to 151px wide to accomodate for Standard and Retina Screens simultaneously.

Updating Skin Styles

This template's styles have been created to make all expected styles easily accessible and updatebale. In the CSS folder you can find the skin-1.css or skin-2.css files. The skin file has been carefully structured so that updating all colors and fonts is as minimal as possible.

When opening the skin files, please note the table of contents in relation to which element you want to update. The skin file is also where all fonts are declared.

Note: Update the hexadecimal colour values to create your own colour scheme.

Layout Options

Boxed or Full Screen

This theme has the ability to be full width or boxed. By default it comes full width, however to change to boxed version simply update the css declaration in your html pages from full.css to boxed.css

Slider Revolution

A premium jQuery plugin is used to control the masthead slider.

Slider Revolution has its own documentation that can be found in the following directory: js/ rs-plugin / documentation / documentation.html

A basic instance of the slider has been implemented in the home pages, if you wish to expand on this with additional transitions, and additional captions, please refer to the Slider Revolution documentation.

Useful Tip:

Slider Revolution has a range of alternative slide transitions that can be implemented.

Theme UI Elements

Included with this template is a styles.html file that covers all the major layout and styling options for columns, form elements, quotes, images, tables and other html features.

The Styles HTML file is well structured to allow you to copy and paste elements you want to use in your standard web pages. The Style page is intended to be used as a master reference file for layout and style options. Ideally you should not edit the Styles file, instead once copied and pasted, perform edits on the website pages you intend to make live.

Home Page Parallax Effect

You can control the speed of the home page parallax section by update the value on data-stellar-background-ratio="1.35". This ratio controls the speed at which the image scrolls. Higher the ratio the faster the speed.

Useful Tip:

Kinda a no brainer but don't include the styles.html file as one of your web pages.

Contact Map

The Google map on the contact page can be easily updated to your own Google map by doing the following.

  • Go to maps.google.com
  • Enter in your desired location
  • Click on the link icon and select - Paste HTML to embed in website
  • Paste the map code snippet into "Outter Wrapper / Feature Map"
  • Update the height attribute on the iframe to suite your own desires, recommended 550 high

Useful Tip:

For a clean sleek map header, Remove the additional link to "View Larger Map" that comes in the code snippet.

Contact Form

The contact form is set up to work on most web servers, the form relies on PHP (as current as possible) being installed on the server in order to operate.

Included with this template is 2 contact form variations, one with recaptcha and one without. See Google Recaptcha

Standard Contact Page

If you choose to use the non-recaptcha contact.html form all that is left for you to do is to update the processForm.php file in the php folder and include your own email address in order to receive emails from the contact form. When updating the processForm.php file look for the following:

// --------------------------------------//
// Send the email // INSERT YOUR EMAIL HERE
$to = "your@email.com";
// --------------------------------------//
Recaptcha Contact Page

If you are using the recaptcha contact.php form you will need to update a couple more items. Firstly go sign up to use Googles recaptcha Service, see link here

Once signup to Recaptcha, and having gone through the process of defining a domain, you will be provided with both Public and Private Keys to use in your web pages.

Now open the processForm-wcaptcha.php file, look for the following /* Enter Private Key Below */ , paste your private key (number) in place of the "*****".

Next in the same processForm-wcaptcha.php file look for the following and update your email address.

// --------------------------------------//
// Send the email // INSERT YOUR EMAIL HERE
$to = "your@email.com";
// --------------------------------------//

Lastly open your contact.php file and look for / Recaptcha Public Key, around line 298, paste in you public key (number) in place of the "*****". Note: the recaptcha contact form will only work once on a web server.

Note:

The contact form is delicate, to avoid errors, please do not edit anything other than this email address unless you know what your doing.

Google Analytics

At the very bottom of the HTML you will find a Google Analytics code snippet to use. Simply replace the UA ID number with your own to set up Google Analytics and start receiving stats and figures about your webpage, or delete this whole section and copy and paste in the code snippet Google provides.

Also DO NOT FORGET to update the title and description of your web page. You can find these at the very top of the HTML file, just look for

<title>Sport</title>
 <meta name="description" content="">
and update them to you own apps name and a description between the content="" for search engines.

Useful Tip:

Google Analytics not only lets you measure sales and conversions, but also gives you fresh insights into how visitors use your site
